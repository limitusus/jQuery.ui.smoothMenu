Description
---

MooToolsのMenuMaticに近い動きを実装したjQueryPluginです。

以下のような機能を実装しています。

* 階層メニュー
* 縦置き、および横置きメニュー
* ThemeRoller
* Enable, Disable, Destroy

Required
---

jQuery v1.4.x or heigher
jQuery UI v1.8.x or heigher
  jquery.ui.core.js
  jquery.ui.widget.js

Usage
---

$('#root > li').smoothMenu();
